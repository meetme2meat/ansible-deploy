# Michef

### README

#### Using ruby 2.2.0

#### System Dependencies
* Postgres (http://coolestguidesontheplanet.com/installing-postgresql-database-os-x-10-9-mavericks-configure-phppgadmin/)

#### Database creation  
* `rake db:create`

#### Database initialization    
* `rake db:migrate`  
* `rake db:seed`

#### How to run the test suite  
* `rake test`
